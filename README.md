# Perfect Planner

Perfect Planner is a simple planning app project. This repository starts with a lightweight HTML and CSS base that can grow into a full daily, weekly, or project planning experience.

## Project Files

- `index.html` - Main app page.
- `style.css` - Base styling for layout, typography, and planner UI.

## Getting Started

Open `index.html` in a browser to view the starter page.

## Next Steps

- Add planner sections and interactions.
- Decide on daily, weekly, and monthly views.
- Add JavaScript when the app needs dynamic behavior.
